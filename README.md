# Geelracafirm

This is a simple Node.js project to run a website.

## Setup

1. Install dependencies:
    ```sh
    npm install
    ```

2. Run the server:
    ```sh
    npm start
    ```

3. Open your browser and navigate to `http://localhost:3000`.
